# Update instructions
The website is made by editing sotsef.html and commiting changes. The changes will appear on the website usually within a minute.

There are two important lists:
 - rawData (line 169 - 191): This contains the xyz coordinates of all the data points in the scatter plot
 - dataNames (line 193 - 216): This contains the unique name associated with each data point. The order is the same as the list.

These should not be edited.


There is one important dictionary, which is the only thing that should be edited:
 - predictors (line 218+): The keys are the pseudonyms of atendees (avoid real names for GDPR). The values are lists of crystals that the the owner of the pseudonyms has found.

If an attendee has already contributed to the landscape, their new data should be added to their existing list. Do not create a new entry for that person.



# Science
The landscape shows the 64 lowest energy Z'=1 crystal structures for Theobromine.
The experimental crystal is Z'=2, but we can keep that to themselves.

Do not tell atendees how many crystals there are (64) and do not tell them which is the global minimum (I haven't even checked).
The real landscape has far more than 64 crystal structures, but we've don't want to have too many posters.
We want to use this as a tool to show that it's very difficult to know when we've found all the crystal structures, and also that it's impossible to know if we've found the lowest energy crystal structure without knowing that we've found all the crystal structures.

If they want to see the crystal structures, I've provided the res files, which can be opened in Mercury.
